🏠 Tenant Verification System

A simple full-stack project to verify tenant identity using document upload, OCR, and basic verification steps.

🚀 Features
📤 Upload Aadhaar / ID documents
🧠 Extract text using OCR (future step)
🤳 Upload selfie for verification
📲 OTP verification (planned)
📄 Basic verification result
🧰 Tech Stack
FastAPI (Backend)
Python
React (Frontend - optional)
Tesseract OCR (future)
OpenCV (future)

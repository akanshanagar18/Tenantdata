🏠 Tenant Verification System

A simple full-stack project to verify tenant identity using document upload, OCR processing (planned), and basic verification steps. The system is designed for rental/PG/hostel use cases to make tenant onboarding faster and more reliable.

🚀 Features
📤 Document Upload
Upload Aadhaar or government ID documents
Secure file handling on backend
🧾 Basic Verification Flow
Tenant submits identity details
Admin/system reviews uploaded data
Verification status shown (Pending / Approved / Rejected)
🧠 OCR (Planned)
Extract text from uploaded ID documents using Tesseract OCR
🧰 Tech Stack
Backend
FastAPI (Python)
Uvicorn
File storage handling
Frontend (Optional / if used)
React.js
Basic form UI
Future Add-ons
Tesseract OCR (text extraction)
OpenCV (image preprocessing)
